Types:
* character `char`
* 64-bit integer `int`
* 32-bit `float`
* boolean `bool`



Expressions:
1. Literals
   1. character `'c'`
   2. int `42`
   3. float `3.14`
   4. bool `true`/`false`
2. Arithmentic operators: `+-*/%`

Statements:
1. Variable definition: `let TYPE NAME = EXPRESSION;`
2. Assignment: `NAME = EXPRESSION;`
3. Return: `return EXPRESSION;`
4. Empty statement `NOOP;`
